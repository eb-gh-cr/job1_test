=== установка

```
установить rvm
bundle
установить редис
rake db:seed
rails s
```

curl -i -H "Accept: application/json" "localhost:3000/api/products"

  --request POST \
  --data '{"product_id":"1","quantity":"1"}' \
  http://localhost:3000/api/cart
